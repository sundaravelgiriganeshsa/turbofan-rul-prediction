# ✈️ Turbofan Engine Remaining Useful Life (RUL) Prediction

## 📌 Overview
This project focuses on predicting the **Remaining Useful Life (RUL)** of turbofan engines using machine learning techniques. The goal is to estimate how many operational cycles an engine can perform before failure.

Predicting failures in advance helps in:
- Preventive maintenance  
- Reducing downtime  
- Improving safety  
- Lowering operational costs  

---

## 📊 Dataset
- Dataset used: **NASA Turbofan Engine Degradation Simulation Dataset**
- Contains multiple sensor readings such as:
  - Temperature  
  - Pressure  
  - Vibration  
- Each engine has multiple cycles until failure
